## ✨ Features

- **⚡ Fast & Efficient**: Quickly generate invoices with automated tax calculations for CGST (9%) and SGST (9%).
- **🧑‍💼 Customer Management**: Dynamically select and add customer GSTINs for faster data entry.
- **📊 Excel Export**: Download and export all your invoice data directly to Excel spreadsheets using SheetJS.
- **☁️ Cloud & Local Management**: Import/Export JSON data, manage cutoff records, and save/load from the cloud.
- **🌙 Modern UI**: A clean, accessible, and intuitive user interface built with semantic HTML and CSS.

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Libraries**: [SheetJS (xlsx)](https://sheetjs.com/) for Excel data processing.

## 📂 Installation & Usage

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/skullgamerss/invoice-system.git
   ```
2. Navigate to the project directory:
   ```bash
   cd invoice-system
   ```
3. Since this is a client-side web application, simply open the `index.html` file in your modern web browser to start using it! No server setup is required.
